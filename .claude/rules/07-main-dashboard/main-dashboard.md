# Main goal of this rule

This is a kind of dashboard/main menu with tiles

- Calendar: takes the user to the calendar main view
- Inbox: inbox tasks `inbox.png`. You can add a new inbox task by tapping the plus button `inbox-new.png`
- Favourites: favourites tasks `favourites.png`. You can add tasks here using the same as `inbox-new.png`
- Past due ("Venciodos"): list of incompleted tasks from the past `past-due.png`. Tapping the three dots menu next to "Anytime" opens the floating window
  like this one `past-due-anytime-menu.png`
