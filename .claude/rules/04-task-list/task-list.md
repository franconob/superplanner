# Main goal of this rule

Display a list of tasks, ordered by creation time. Each task can be marked as checked.
Tasks can be filtered too.
You can add a new inline task by pressing the plus button. If you tap on the information icon on the right
it should open the add new task modal

This task list is opened by tapping on the inbox icon on the home screen (top right corner)
